# Spider2 - Project
![BuildStatus](https://github.com/Vitrix-Software-s-r-o/spider2/actions/workflows/build-run-tests.yml/badge.svg)

The aim of this project is to create library for creation HTTP servers in composable manner in moder C++ using Boost.Asio and Boost.Beast with full coroutines support.

Check out the [documention](https://spider2.vitrix.cz/)
